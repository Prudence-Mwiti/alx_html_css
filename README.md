# CSS Advanced – ALX Project

This project focuses on applying advanced CSS styling to build a visually accurate website based on a provided Figma design. No external frameworks or libraries are allowed — only HTML, CSS, and a little JavaScript.

## Objectives
- Understand CSS structure and selectors
- Apply styles to elements, sections, and components
- Learn the CSS Box Model
- Create a clean, responsive layout using only CSS
- Match styles from a Figma design prototype

## Files
- `index.html` – HTML structure
- `styles.css` – CSS styling
- `images/` – Project images
